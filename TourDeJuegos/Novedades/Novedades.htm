Torneos, ligas, tutoriales, todo sobre juegos, apúntate registrándote en el foro gratis.

 

 

En este apartado, Novedades, conforme vaya avanzando la web iremos posteando aquí y en el foro las actualizaciones de la pagina web y foro, torneos/concursos, sistema de puntuación, forma de jugar, reglas, etc.

 

Se recompensará con puntos por participar y ganar torneos que esos puntos se podrán utilizar para comprar cosas del juego, como mejorar la plantilla fichando jugadores si es un juego de fútbol o también para intercambiarlos por dinero de juego que servirá para más cosas que para comprar jugadores, cómo cambiar de equipo y más cosas.

 

En esta página web queremos decir que todos somos importantes nadie mas/menos que nadie todos tendrán las mismas oportunidades y se les juzgará de igual manera.

Lo único que se pide será seriedad para que se pueda llevar a cabo buenos torneos y ligas.
